MySQL无法连接

发表于 2024-10-02 更新于 2024-10-07 分类于 Problem 本文字数： 406 阅读时长 ≈ 1 分钟

今天无意中点开了mysql客户端，发现输入密码后闪退，感觉是密码输错了。于是win+R调用命令行cmd输入mysql -u root -p打开Mysql，试了几次可能的密码都登录不上，报错ERROR 2003 (HY000): Can’t connect to MySQL server on ‘localhost:3306’ (10061)，无法连接到Mysql。随后又打开navicat，依然无法连接。有点小慌，害怕是刚刚清理电脑C盘文件时误删除了什么东西。于是赶紧去google，找到一篇CSDN上的博客，第一步就解决了我的问题。

ERROR 2003 (HY000): Can’t connect to MySQL server on ‘localhost’ (10061)-CSDN博客

简述：在开始处搜索计算机管理，点击服务，找到Mysql，右键启动，再重新登录Mysql，就可以了！

猜测大概是长时间不用Mysql，系统自动停止了服务，需要手动启动。
